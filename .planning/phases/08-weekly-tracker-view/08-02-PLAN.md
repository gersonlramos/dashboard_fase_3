```python
    # ── Impedimentos ────────────────────────────────────────────────────────────
    st.markdown("---")
    st.subheader("🚧 Impedimentos")

    _imp_path = os.path.join(DADOS_DIR, "pendencias_BF3E4-293.csv")
    try:
        _df_imp = pd.read_csv(_imp_path, encoding='utf-8', sep=',')
    except FileNotFoundError:
        st.warning(f"Arquivo não encontrado: {_imp_path}")
        _df_imp = pd.DataFrame()

    if not _df_imp.empty:
        # Filter open items (status not in done list)
        _imp_status_done = {s.lower() for s in _status_done}
        _df_imp_open = _df_imp[
            ~_df_imp['Status'].astype(str).str.strip().str.lower().isin(_imp_status_done)
        ].copy()

        if _df_imp_open.empty:
            st.success("Nenhum impedimento em aberto.")
        else:
            # SLA: vencido when Deadline < today
            _df_imp_open['_deadline_ts'] = pd.to_datetime(
                _df_imp_open['Deadline'], dayfirst=True, errors='coerce'
            )
            _df_imp_open['Vencido'] = _df_imp_open['_deadline_ts'] < _hoje
            _df_imp_open['SLA'] = _df_imp_open['Vencido'].map(
                {True: '⚠️ Vencido', False: '✅ OK'}
            )

            # Display columns (drop internal helper columns)
            _imp_cols = [c for c in ['Chave', 'Titulo', 'Status', 'Prioridade', 'Deadline', 'SLA']
                         if c in _df_imp_open.columns]
            st.dataframe(
                _df_imp_open[_imp_cols],
                use_container_width=True,
                hide_index=True,
            )

            _vencidos = int(_df_imp_open['Vencido'].sum())
            if _vencidos:
                st.error(f"⚠️ {_vencidos} impedimento(s) com SLA vencido.")
    else:
        st.info("Dados de impedimentos indisponíveis.")
```

- `_status_done` and `_hoje` are already in scope (module-level, defined before the tab dispatch).
- `pd` is already imported.
- The `_df_imp_open` variable name avoids any collision with the existing `df_imp` name used in the "⚠️ Impedimentos" tab (check with `grep "df_imp" app/dashboard/dashboard.py` — use a distinct prefix `_df_imp_open` throughout).
- `dayfirst=True` is required: the `Deadline` column in `pendencias_BF3E4-293.csv` uses day-first format (e.g., `25/03/2026`).

Append the following snippet (same 4-space indentation):

```python
    # ── Gráfico: Progresso vs. Meta Semanal ─────────────────────────────────────
    st.markdown("---")
    st.subheader("📊 Progresso vs. Meta Semanal por Lake")

    # "Concluídos" = total items - remaining (proxy for done count at lake level)
    # Derive total per lake from df to avoid needing a separate total column
    _total_por_lake = {}
    for _lk2 in _df_sem['lake'].tolist():
        _mask2 = df['Data-Lake'].astype(str).str.strip().str.upper() == _lk2
        _total_por_lake[_lk2] = int(_mask2.sum())

    _df_sem['total_itens'] = _df_sem['lake'].map(_total_por_lake).fillna(0).astype(int)
    _df_sem['concluidos']  = (_df_sem['total_itens'] - _df_sem['itens_restantes']).clip(lower=0)

    # meta_semanal = velocidade_necessaria * 5 business days (rounded to int)
    _df_sem['meta_semanal'] = (_df_sem['velocidade_necessaria'] * 5).round(0).astype(int)

    _fig_bar = go.Figure()

    _fig_bar.add_trace(go.Bar(
        name='Concluídos (total)',
        x=_df_sem['lake'].tolist(),
        y=_df_sem['concluidos'].tolist(),
        marker_color='#2ecc71',
    ))

    _fig_bar.add_trace(go.Bar(
        name='Meta Semanal',
        x=_df_sem['lake'].tolist(),
        y=_df_sem['meta_semanal'].tolist(),
        marker_color='#e74c3c',
    ))

    _fig_bar.update_layout(
        barmode='group',
        title='Concluídos vs. Meta Semanal por Lake',
        xaxis_title='Lake',
        yaxis_title='Itens',
        template=plotly_template,
        paper_bgcolor=plotly_paper_bgcolor,
        plot_bgcolor=plotly_plot_bgcolor,
        font=dict(color=plotly_font_color),
        legend=dict(orientation='h', yanchor='bottom', y=1.02, xanchor='right', x=1),
        xaxis=dict(**plotly_axis_style),
        yaxis=dict(**plotly_axis_style),
    )

    st.plotly_chart(_fig_bar, use_container_width=True)
```

Implementation notes:
- `_df_sem` is the local copy created earlier in the `elif` block (by Plan 08-01). Reuse it directly — do NOT call `_calcular_semaforos` again.
- `go` (`plotly.graph_objects`) is already imported in `dashboard.py` (verify with `grep "import plotly.graph_objects" app/dashboard/dashboard.py`).
- `plotly_template`, `plotly_paper_bgcolor`, `plotly_plot_bgcolor`, `plotly_font_color`, and `plotly_axis_style` are all defined at module scope (lines 75–84). No hardcoded hex values for background or text — use these variables only.
- The two marker_color values (`#2ecc71`, `#e74c3c`) are bar fill colours for the data series themselves, not background/text — these are intentional fixed colours for semantic meaning (green = done, red = goal not met) and are acceptable.
- `df` (unfiltered FASE_3.csv) is at module scope.
