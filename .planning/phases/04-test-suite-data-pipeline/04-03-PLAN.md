```python
def adf_para_texto(node):
    """Converts Atlassian Document Format (ADF) to plain text.
    - None → ""
    - str → str (returned as-is)
    - list → "".join(adf_para_texto(item) for item in list)
    - dict, type="text" → node["text"]
    - dict, type="hardBreak" → "\n"
    - dict, type="paragraph" or "heading" → joined children + "\n"
    - dict, type="bulletList" → "• {item}\n• {item}\n..."
    - dict, type="orderedList" → "1. {item}\n2. {item}\n..."
    - dict, any other type → joined children (no extra newline)
    """
```

Note: `script_pendencias.py` is safe to import — all HTTP calls happen inside `main()` which is guarded by `if __name__ == '__main__':`.
